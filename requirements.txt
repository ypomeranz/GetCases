# GetCases runtime dependencies.
#
# Core (case search, opinion text, statute/regulation viewer):
requests
beautifulsoup4

# In-app PDF viewer (View PDF button in the opinion window) and PDF-brief text
# extraction ("Open Brief…" reads a PDF's text layer via pypdfium2).  Optional —
# without these the app still runs; the opinion viewer offers the browser and
# PDF briefs report that the text layer can't be read.
pypdfium2
Pillow

# "Open Brief…" reads .docx and .rtf with the standard library (no extra
# package needed).  Legacy .doc files are read via an external converter when
# one is installed — any of `antiword`, `catdoc`, or LibreOffice (`soffice`) —
# otherwise a built-in best-effort text scan is used.  These are system tools,
# not pip packages.

# English Reports citations ("156 Eng. Rep. 145") link to the free CommonLII
# scan.  Detection + the shipped index (eng_rep_index.tsv.gz) are offline and
# need nothing extra.  Viewing the PDF *inside* the app is optional: CommonLII
# is behind CloudFlare, so the user passes the "Just a moment..." check once and
# the app fetches the scan.  Two ways to clear it:
#   * playwright — pass the check in the user's own browser (Chrome/Edge via
#     channels, so no browser download); the clearance is then saved and reused.
#   * curl_cffi (+ Firefox) — read Firefox's clearance cookie and fetch with a
#     browser-matching TLS fingerprint.  browser_cookie3 is an optional fallback
#     for reading that cookie.
# Without any of these (or a browser to drive), E.R. citations simply open in
# the user's browser instead.
playwright
curl_cffi
browser_cookie3
